#HackBio NGS Internship 2025 – Summary

Completed hands-on training in NGS data analysis across three stages, culminating in a research-focused final project.

Internship Stages & Skills

Stage 0 – Linux & Bash Fundamentals

- Command-line proficiency, file handling, text parsing, terminal management
- Conda environment setup and installation of bioinformatics tools

Stage 1 – Whole-Genome Sequencing (WGS)

- QC, alignment, variant calling, and annotation
- Interpretation of pathogenic variants using ClinVar and other databases

Stage 2 – RNA-seq Analysis

- QC, trimming, alignment, gene counting
- Differential expression analysis with DESeq2
- Functional enrichment (GO, KEGG) and data visualization

Final Project – lncRNAs as Biomarkers for Cancer Immunotherapy

- Upstream processing, alignment, and annotation in Bash
- Differential expression analysis using DESeq2
- Co-authored team manuscript

Overall Skills Gained

- Linux command-line & Bash scripting
- NGS workflows: WGS & RNA-seq
- Variant analysis, differential expression, functional enrichment
- Bioinformatics software management and collaborative research
